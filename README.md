## Descrição

Tea store.

## Instalação

```bash
$ pnpm install
```

## Executar o aplicativo

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Documentação swagger

Após executar o aplicativo, abra o localhost no endpoint */api*

```bash
# exemplo
$ localhost:3000/api
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
